# 《最后的咒语》Demo：游戏规则与数值表

> 状态：以当前 `index.html` 的实际实现为准整理。
> 本文档同时是游戏数值的来源文件。最后更新：2026-07-22。

## 1. 如何使用这份文档

### 阅读规则

本文档前半部分记录当前已经实现的完整规则，以及尚未实现的边界。以后忘记规则时，优先查看本文档。

### 修改数值并同步到游戏

1. 打开本文档，找到最后的“可编辑数值区”。
2. 只修改 JSON 冒号右侧的数字或引号内文字，不要删除括号、逗号和字段名。
3. 保存文档。
4. 双击项目中的 `同步规则到游戏.bat`。
5. 刷新 `index.html`，修改后的数值就会生效。

执行 `npm run build` 时也会自动同步本文档。自动生成的 `game-config.js` 不应手工修改。

适合直接调整：地图尺寸、单位名称、职业、位置、攻防血、先攻、行动力、技能倍率、技能范围、击退距离、AI移动速度等。

不适合只改文档：新增技能机制、新增伤害类型、改变AI决策方式、加入装备或状态效果。此类改动需要同时修改程序逻辑。

## 2. 游戏概述

- 类型：网页端2.5D上帝视角战棋Demo。
- 棋盘：等距视角方格地图，默认60×60格。
- 比例：每格代表1米。
- 阵营：3名我方单位对6名敌方单位。
- 基本节奏：所有存活单位依次完成一次行动，构成一个完整回合。
- 当前目标：验证棋盘比例、移动、技能、伤害、回合顺序和基础敌人AI。

## 3. 地图与占位规则

- 所有单位和障碍物占据整数格子。
- 单位不能与另一名存活单位处于同一格。
- 单位不能进入障碍物格子或地图边界外。
- 移动只允许上、下、左、右，不允许斜向移动。
- 当前地图有5组矩形障碍物，具体坐标见文末数值区。
- 地图支持鼠标滚轮缩放、拖拽平移、显示或隐藏网格。
- 初始美术比例约定为Q版1:2头身比、角色身高1.2米；当前仍是抽象白模表现，没有碰撞体高度和高低差规则。

## 4. 回合与行动顺序

1. 只将仍有生命的单位加入当前回合队列。
2. 按先攻值由高到低排序。
3. 先攻相同时，我方单位排在敌方单位之前。
4. 同阵营且先攻相同时，按内部单位ID排序。
5. 当前单位结束行动后，轮到队列中的下一名单位。
6. 所有存活单位各行动一次后，回合数加1并重新生成行动队列。
7. 单位死亡后会从地图、列表和后续行动中移除。

当前所有我方单位先攻为10，所有敌方单位先攻为5，所以每回合默认由3名我方依次行动，然后6名敌方依次行动。

## 5. 行动力与移动

- 每移动1格消耗1点行动力。
- 我方单位每回合开始时恢复至12点行动力。
- 敌方单位每回合开始时恢复至10点行动力。
- 可移动范围使用实际寻路计算，会绕开障碍物和其他存活单位。
- 玩家可以在回合内多次移动，行动力按每次实际路径长度累计扣除。
- 攻击不会自动结束行动。攻击后仍可使用剩余行动力移动。
- 只有玩家主动点击“结束行动”后，当前单位才完全停止行动。

### 重置移动

- 每个单位行动开始的位置会记录为“回合起点”。
- 攻击前可以点击“重置移动”，回到回合起点并恢复全部行动力。
- 一旦已经发动攻击，本回合不能再重置移动。
- 重置移动只撤销位置和行动力，不是整局重置。

## 6. 攻击与伤害

### 当前伤害公式

`最终伤害 = max(最低伤害, round(攻击力 × 技能倍率) − 目标防御力)`

- 当前最低伤害为1。
- 技能倍率先与攻击力相乘并四舍五入，再减去防御。
- 当前物理伤害和魔法伤害都使用同一防御属性，没有分别计算物防和魔防。
- 每名我方单位每回合最多攻击1次。
- 受伤时在目标上方显示红色浮动伤害数字。
- 当前没有暴击、命中率、闪避、减伤率、护盾、治疗、反击和状态效果。

### 当前示例伤害

敌人防御为5、我方攻击为10时：

| 技能 | 计算 | 最终伤害 |
|---|---:|---:|
| 我高·直接攻击 | round(10×100%)−5 | 5 |
| 我高·击退攻击 | round(10×80%)−5 | 3 |
| 我强·魔法球 | round(10×80%)−5 | 3 |
| 我强·魔力降临 | round(10×50%)−5=0，触发最低伤害 | 1/每名敌人 |

## 7. 技能规则

### 我高：战斗法师（近战）

#### 技能1：直接攻击

- 目标：单个敌人。
- 施法范围：上下左右直线1格。
- 伤害：100%攻击力的物理伤害。
- 不可斜向攻击。

#### 技能2：击退攻击

- 目标：单个敌人。
- 施法范围：上下左右直线2格。
- 伤害：80%攻击力的物理伤害。
- 命中后尝试将目标沿远离施法者的方向击退2格。
- 地图边界、障碍物或其他存活单位会阻止击退；无法走完时只击退实际可移动的格数。

### 我帅：盾卫

- 当前只有基础属性和移动能力。
- 尚未配置技能。

### 我强：远程法师

远程技能的施法范围按“横向格数差＋纵向格数差”计算，即曼哈顿距离。不能用斜线缩短距离。

#### 技能1：魔法球

- 目标：施法距离9格内的单个敌人。
- 伤害：80%攻击力的魔法伤害。
- 不是弹道技能，不产生飞行路径碰撞。
- 障碍物和其他单位不会阻挡魔法球。

#### 技能2：魔力降临

- 类型：落点型范围技能。
- 可以选择施法距离9格内的任意落点，包括空地和自己所在格。
- 以落点为中心，影响3×3共9格区域。
- 对区域内所有存活敌人分别造成50%攻击力的魔法伤害。
- 不伤害我方单位。
- 选择技能后，红色区域表示可选落点；鼠标悬停时橙色区域预览3×3效果范围。

## 8. 敌人AI

- 敌人按照回合队列一个一个行动，不会同时移动。
- 敌人会寻找“实际可走路径最近”的存活我方单位。
- AI使用上下左右寻路，会绕开地图边界、障碍物和其他存活单位。
- 敌人逐格移动，每格消耗行动力，并实时显示剩余行动力。
- 敌人移动到目标相邻格，或行动力耗尽后结束行动。
- 如果无法抵达任何目标的相邻格，会移动到当前能够到达且最接近我方的格子。
- 当前敌人AI只会追击和移动，不会自动攻击。
- 整局重置会取消尚未完成的旧AI行动，避免重复执行。

## 9. 界面提示

- 我方单位、卡片、生命条和移动区域使用绿色。
- 敌方单位和卡片使用红褐色。
- 当前行动单位有选中描边。
- 技能可施法格使用红色。
- 魔力降临的3×3预览使用橙色。
- 左侧可以切换查看我方或敌方单位列表。
- 右侧显示当前单位属性、行动力、技能和战斗提示。

## 10. 整局重置

点击顶部“重置演示”后：

- 所有单位回到初始坐标。
- 所有单位生命和行动力恢复到初始值。
- 回合数回到第1回合。
- 当前行动回到行动队列第一名。
- 镜头缩放和平移恢复默认值。
- 浮动伤害数字和正在执行的敌人AI被清除或失效。
