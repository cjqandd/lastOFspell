# 网页游戏：跨电脑交接说明

## 给亲戚直接玩

把 `亲友试玩-最后的咒语.html` 发给对方即可。它是完整的单文件网页，不需要安装软件、联网或发布到网站；下载后双击，用浏览器打开即可游玩。

## 在另一台电脑继续制作

1. 把 `最后的咒语-跨电脑同步包.zip` 拷到移动硬盘。
2. 在另一台电脑解压，得到项目文件夹。
3. 修改 `index.html`，这是游戏的主源码。
4. 需要更新发布版本时，在项目文件夹中运行 `npm run build`。

## 同步约定

不需要每次小改动都打包。准备换电脑时告诉 Codex「重新同步/重新打包这个项目」，即可生成一份新的携带包。每次请用新包完整覆盖另一台电脑中同名项目文件夹，避免把两边不同版本混在一起。
